Sentinel
=========

Fully custom scouting data processing, analytics, and display backend
written in `flask <https://flask.palletsprojects.com/en/stable/>`__ and
packaged with `grafana <https://grafana.com>`__ in
`docker <https://www.docker.com>`__.
